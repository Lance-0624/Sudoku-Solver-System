# Sudoku-Solver-System
A modular Sudoku solver implemented in Java, featuring decoupled Model-View architecture and various design patterns.

## Key Features
- **Design Patterns**: Implemented Strategy pattern for various solving algorithms, Decorator pattern for functional enhancements, and Factory Method for logical decoupling from the view.
- **Modular Architecture**: Features a clear Model-View-Controller (MVC) separation, ensuring business logic is fully independent of user interaction.
- **Functional Verification**: Includes a comprehensive Command Line Interface (CLI) for efficient Sudoku board generation and automated solving.

## Tech Stack
- **Language**: Java
- **Architecture**: MVC (Model-View-Controller)
- **Design Patterns**: Strategy, Decorator, Factory Method

## How to Run
1.Ensure a Java development environment is installed.

2.Compile and run Main.java to launch the CLI.
